解析

連続する素数 p1=19, p2=23 の場合を例にして考える。

23n が10k + 9` と書ける場合、

3 * n = 9 (mod 10)
=> 23n = 23(10k + 3) = 230k + 69

である。

230k + 69 が 100m + 19 と書ける場合、

3 * k + 6 = 1 (mod 10)
=> 230k + 69 = 230(10m + 5) + 69 = 2300m + 1219

よって、S = 1219 である。

解法

n, m について以下を満たす x を、x(n, m) と書く。

n * x = m (mod 10)

x_i は x の i 桁目を表すとする。 以下の漸化式により、S を求めることができる。 ここで、 n は p1 の桁数である。

a[1] = x(p2, p1_1)
b[1] = a[1] * p2
a[i] = x(p2, p1_i - b[i-1]_i)
b[i] = a[i] * p2 * 10^(i-1) + b[i-1]
S = b[n]

例 p1=19 と p2=23 の場合

a[1] = x(23, 9) = 3
b[1] = 3*23 = 69
a[2] = x(23, 1 - 6) = 5
b[2] = 50*23 + 69 = 1219
S = b[2] = 1219

b[1] = 7*13 = 91
a[2] = x(13, 1 - 9) = 4

S = b[2] = 611
